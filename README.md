# Book Reader

A Flutter Android book reader app supporting EPUB, PDF, and TXT formats.

## Prerequisites

- Flutter 3.22+ (`flutter --version`)
- Android SDK with API 26+
- A device or emulator running Android 8.0 or higher

## First-time setup

This repo contains only the `lib/`, `android/app/src/main/AndroidManifest.xml`, `pubspec.yaml`, and config files. The platform scaffolding is generated by Flutter itself. From an empty folder:

```bash
# 1. Initialize the Flutter project (only needed once per clone if android/ios folders are missing)
flutter create --org com.yourname --project-name book_reader --platforms=android .

# 2. Pull dependencies
flutter pub get

# 3. Replace the auto-generated AndroidManifest.xml with the one in this repo
#    (it has the storage permissions we need)

# 4. Run on a connected device
flutter run
```

If you cloned this repo and the `android/` folder is already complete, just:

```bash
flutter pub get
flutter run
```

## Project structure

```
lib/
├── main.dart              # Entry point
├── app.dart               # Root widget, theme, router
├── core/
│   ├── theme/             # Light/dark/sepia themes + provider
│   ├── database/          # SQLite setup
│   └── router/            # go_router config
├── features/
│   ├── library/           # Book list, import, storage scan
│   ├── reader/            # EPUB/PDF/TXT viewers
│   └── settings/          # User preferences
└── shared/                # Reusable widgets
```

## State management

[Riverpod 2.x](https://riverpod.dev) — providers live next to the feature they belong to.
